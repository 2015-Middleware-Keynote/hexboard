A hexboard for visualizing Kubernetes pod status in OpenShift

Thousand containers:

  • A live visualisation of the state changes pulling data from /api/v1beta3/watch/pods.
  • A visualisation of the pre-started pods pulling data from /api/v1beta3/watch/pods.
  • A playback of the of the state changes using real data stored in a file.
  • A visualisation of random state changes to show an optimal use case.
